These are the common building blocks I have pullled from my research so far.

Core Testing
  • Load Testing - Offer loading testing services for HTTP endpoints.
  • Response Header Inspector - Ability to inspect a response header.
  • Response Body Inspector - Ability to inspect a response body.
  • Request Retry - Ability to retry a specific request.
  • Request Sharing - Allow for the sharing of requests.
  • Request Playback - Enable the playback of a single request.
  • Request Scheduling - Allow for the scheduling of requests at a specific time.
  • Request Compare - Allow for the comparing of two requests.
  • Request Scripting - Provide the ability to script against requests.
  • Request Automation - Allow for the automation of multiple requests.
  • Request Commenting - Enabling users to comment on a specific request.
  • Simulator - Provide tools for simulating requests.
  • Templates - Offer templates of common or specialized requests.
  • Data Scenarios - Provide specific data scenarios that can be executed.

Targeted Testing
  • Region Based Testing - Allow for testing from a specific region.
  • Provider Based Testing - Allow for testing from a specific provider

Authentication
  • Basic-Auth - Use Basic Auth for authentication.
  • API Keys - Use API keys for authentication.
  • OAuth - Use OAuth for authentication.

Utility
  • Collections - Allow tests to be organized into collections.
  • Command Line Interface - Provide a command line interface for working with tests.
  • Virtualization - Allow for tests against virtualized APIs.
  • Teams - Allow for team interaction when testing.
  • API - Provide an API for all parts of testing platform.

Import
  • Postman - Allow for the importing of APIs with Postman.
  • Swagger - Allow for the importing of APIs with Swagger.
  • RAML - Allow for the importing of APIs with RAML.
  • HAR - Allow for the importing of APIs with HAR
  • JUnit XML - Allow for the importing of APIs with JUnit XML.

3rd Party
  • JMeter - Provide 3rd party integration with JMeter.
  • Selenium - Provide 3rd party integration with Selenium.
  • Jenkins - Provide 3rd party integration with Jenkins.
  • TeamCity - Provide 3rd party integration with TeamCity.
  • Bamboo - Provide 3rd party integration with Bamboo.
  • Travis - Provide 3rd party integration with Travis.
